文件目录管理命令

Created2023-11-28|Updated2023-11-28
|Post Views:

touch 创建空白文件或者设置文件时间

 touch -a   -m   -d         atime   mtime
 访问时间   修改时间    同时修改
 touch -d "20200202 12:00" xxx.cfg

mkdir

mkdir hellodirctory     创建单目录
mkdir -p a/b/c/d/e      递归创建目录

cp

cp a b      将a文件复制到b处

mv 剪切或者重命名文件

mv a b      将a文件剪切到b处。如果ab在同一个文件夹下，a重命名成了b

rm remove

删除文件或目录
rm -f   -i  -r  -v
强制删除    删除前询问     删除目录     显示过程

dd

dd if=/dev/zero of=new_file count=10 bs=50M
dd if=/dev/zero of=new_file count=1 bs=500M
dd if=/dev/cdrom of=newos.iso

file 查看文件类型

tar

tar -c  -x  -z  -v  -f
压缩    解压    用gzip格式  显示过程    目标文件名
tar -c bb aa    将aa压缩成bb
tar -x aa bb    将aa解压到bb
