查找定位文件命令

  1. pwd print working directory

  2. cd change directory

    cd - 返回上一次的目录

    cd ~ (username) 返回家目录，类似游戏中的回城

  3. ls list 显示目录中的文件信息
    ls -al
    ls -ld /etc 查看目录的属性

  4. tree 树状图查看目录结构和信息

  5. find [查找范围]]寻找条件
    find /etc -name “host*”

    find /etc -user “hj”

    find / -user hj -exec cp -a {} /root/findresult\

  6. locate 按照文件名称快速搜索到文件所在的位置

    首次使用先生成索引数据库

    updatedb

    ls -l /var/lib/mlocate/mlocate.db

    locate whereis

    locate效率高，find更准确

  7. whereis 同locate先updatedb,搜索命令所在的位置和帮助文档

  8. which 同whereis ,只是搜寻命令本身所在的位置
