编写简单的脚本

vim example.sh
#!/bin/bash
# for example by linuxprobe.com
pwd
ls -al

执行方式1、2

1   bash example.sh
2   ./example.sh
    (chmod u+x example.sh)

接收用户的参数

#!/bin/bash
# for example by
echo "当前脚本的名称为$0 \$0)"
echo "t他一共有$#\$#个参数,分别为$*\$*"
echo "第一个参数是$1\$1"

[root@linuxprobe blog]# bash example.sh a b c d e
当前脚本的名称为example.sh $0)
他一共有5$#个参数,分别为a b c d e$*
第一个参数是a$1

判断用户的参数

$?  返回上一个命令的返回值  返回0为真，其他为假
[ -d /etc/xxx ]  是否为文件夹
[ -e /etc/xxx ]  是否为空，是否存在
[ -f /etc/xxx ]  是否是普通文件
[ -r /etc/xxx ]  当前用户是否可读
[ -w /etc/xxx ]  可写
[ -x /etc/xxx ]  可执行
 逻辑语句   &&  ||  ！  注意非的！要写在整个语句的最前面

整数的比较运算符

-eq     等于
-ne     不等于
-gt     大于
-lt     小于
-ge     大于或等于
-le     小于或等于

字符串的比较运算符

=       相同
!=      不同
z       为空

流程控制语句

if      for     while       case

计划任务服务程序

at 
crontab
分  时  日  月  星期    命令
1.所有命令必须绝对路径  whereis
2.分必须有数值，且不能为空或*   */2 表示每2分钟 
3.日和星期 字段不能同时使用，有冲突

25 3 * * 1,3,5 /usr/bin/tar -czvf backup.tar.gz /home/wwwroot
#每周的135的3：25分用tar命令 将数据打包到home
